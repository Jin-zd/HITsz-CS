选中 $A = 10$ 的概率: 
\[\frac{1}{20},\]
选中 $B = 20$ 的概率: 
\[\frac{1}{10},\]
选中 $A = 10\; \text{and}\; B = 20$ 的概率: 
\[\frac{1}{20} \times \frac{1}{10} = \frac{1}{200},\]
故选中 $A = 10\; \text{or}\; B = 20$ 的概率: 
\[\frac{1}{20} + \frac{1}{10} - \frac{1}{200} = \frac{29}{200},\]
大小为: 
\[10000 \times \frac{29}{200} = 1450.\]
